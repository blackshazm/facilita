# FACILITA - Marketplace de Serviços Domésticos

A **FACILITA** é uma plataforma moderna e segura para contratação de profissionais de manutenção residencial (Elétrica, Hidráulica, Reparos, Pintura) sob demanda.

## 🚀 Tecnologias

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Linguagem**: TypeScript
- **Estilização**: Tailwind CSS
- **Banco de Dados & Auth**: Supabase
- **UI Components**: Radix UI / Lucide Icons
- **Design System**: Emerald (Custom 60-30-10 palette)

## 🎨 Design System: Emerald Green

O projeto utiliza um sistema de cores focado em legibilidade e profissionalismo:

- **60% (Fundo)**: `#EAF0F6` (Cool White)
- **30% (Cards/Superfícies)**: `#FFFFFF` (Pure White)
- **10% (CTAs/Destaques)**: `#10B981` (Emerald Green)
- **Fonte**: Inter (Google Fonts)

## 🛠️ Como Iniciar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Configure as variáveis de ambiente no `.env.local`:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=seu-url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=sua-chave
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

4. Acesse [http://localhost:3000](http://localhost:3000).
